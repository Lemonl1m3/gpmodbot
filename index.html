The Project :: gpmodbot

The Google Play Music on Discord Bot. This project utilizes several third-party and unofficial libraries in order to provide an interactive bot for Discord channels with a link to an All Access Google Play Music account.

Built On Python

The project is built from the ground up using Python, including the gmusicapi and discord.py libraries.

All Access Compatible

Whether you're using a free Google Play Music account or pay for All Access, you will be able to listen to any song your account normally can access.

Configurable Design

When the bot is started, it uses configurations readily accessible in a config.json file. It will also save changes made by any user with permissions.

Service Access

Like songs, create and modify playlists, and let others participate. You can let anyone play songs or keep a playlist looping, it's up to you.
